This folder contains the scripts working for the aircraft GBS3_trimmed2

### `GBS3_trimmed2_script.xml` trims the aircraft

### `GBS3_elevzero2.xml` trims and then set elevator deflection to 0

### `GBS3_elevdoublet2.xml` trims and then runs elevator doublet

### `GBS3_elevchirp2.xml` trims and then runs elevator chirp

Same for files with `ail` instead of `elev`, but with ailerons.
