# Volantex-JSBSim

### `aircraft/`
This folder contains the aircraft model files.

- `GBS3/` → files for the **GBS3** aircraft model (original)
- `c172x/` → files for the **Cessna 172** aircraft model

### `engine/`
This folder contains engine-related files used by the aircraft models.

### `scripts/`
This folder contains simulation and test scripts.

- `scripts/c172x/` → test files for the Cessna 172 model
- `scripts/GBS3/` → test files for the GBS3 model
- 
#### 'GBS3/fail'
This folder contains the test scripts running with the original configuration where trim fails

#### 'GBS3/trim'
This folder contains the aircraft,init,engine, systems and test scripts running with different configurations where trim success
